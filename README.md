## Features

- Character progression tracking
- Daily Arcane/Sacred Symbol tracking
- Estimated time to max Symbol level
- Link Skill reference and descriptions
- Legion block validation and optimization
- Daily and weekly boss tracking
- Weekly boss planning tools
- Character income estimation
- Aggregate income per world
- Genesis and Destiny Liberation tracking
- Liberation schedule forecasting and completion estimates

## Running

Clone this repository:

```bash
git clone https://github.com/Hideyoshizane/MapleTrack.git
```

Acess the folder

```bash
cd MapleTrack
```

Install dependencies

```bash
npm install
```

Execute the application

```bash
npm run dev
```

Server should start at door: 3000
